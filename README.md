# Go Docker App

Simple Go application dockerized.

## Run Locally

1. Build Docker image:
```bash
docker build -t my-go-app .

```

2. Run Docker container
```bash
docker run -p 8080:8080 my-go-app
```

## Docker Hub

Pull the image from Docker Hub:
```bash
docker pull mariammohamed9/my-go-app
```
